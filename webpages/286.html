Further template category notes

This category contains pages in the template namespace. It should not be used to categorize articles or pages in other namespaces.

To add a template to this category:

  • If the template has a separate documentation page (usually called "Template:template name/doc"), add
  •  [[Category:User block templates]]
  • to the <includeonly> section at the bottom of that page. Otherwise, add
  •  <noinclude>[[Category:User block templates]]</noinclude>
  • to the end of the template code, making sure it starts on the same line as the code's last character.
This is a maintenance category. It is used for maintenance of the Wikipedia project and is not part of the encyclopedia. It contains pages that are not articles, or it groups articles by status rather than subject. Do not include this category in content categories.
Wikimedia Commons has media related to User block templates.
Shortcut
  • CAT:UBT

This category gets block templates used by administrators. Many of these templates are heavily used; please be careful when making changes to these. These templates should generally be placed on the user talk page using {{subst:TemplateName}} (some templates may have optional parameters).

When used, these templates should be substituted, except when explicitly suggested otherwise as with {{schoolblock}}, for example.

For non-block 'uw-…' templates, see category:Standardised user warning templates

This is a hidden category. It is not shown on its member pages, unless the corresponding user preference 'Show hidden categories' is set.

This category is used to track images that have had their non-free use rationales assessed via bot. These files will also appear in the category for images with NFUR stated. Humans should assess the accuracy of these non-free use rationales and remove the "auto" parameter if valid.
